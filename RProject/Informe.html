2. Resolucin

2.1 Identificar el probla y objetivos del anlisis

Los suicidios son uno de los principales problemas de la sanidad pblica presente en todos los pases del mundo independientemente de su nivel econmico, poblacin o ubicacin geogrfica. Con este estudio pretendemos desevelar que sectores de la poblacin estn ms en riesgo considerando su edad, sexo, renta per capita y continente usando la informacin de distintos pases a lo largo de los aos.

La finalidad del anlisis es generar un informe sobre el risgo de cada grupo de poblacin con el fin de ayudar a enfocar futuros proyectos a identificar las causas del suicidio en los grupos ms desfavorecidos

Para ello mostraremos y compararemos los distintos grupos de poblacin en base a: + Su sexo + Su gdb_per_capita + Su ao + Su HDI

Adems crearemos un modelo regresivo con los objetivos de: + Completar los datos de HDI for year

2.2 Descripcin del dataset

El siguiente dataset se ha construido a partir de distintos conjuntos de datos (referencias en la bibliografia) para su anlisis. Se compone de 27820 registros con 12 campos. Los registros estn almacenados en un fichero CSV llamado suicides extado de la web Kaggle.

Estos son los campos que contiene:

  • country: Nombre del pas
  • year: Ao del registro
  • sex: Genero de la poblacin
  • age: Grupo de edad de la poblacin del pas ese ao
  • suicides_no: Nmero de suicidios en el pas ese ao
  • population: Habitantes del pas ese ao
  • suicides/100k pop Numero de suicidios de la poblacin por cada cien mil habitantes
  • country-year: Identificador compuesto por los campos pas y ao
  • HDI for year: (Human Development Index), es un indicador del desarrollo humano en base a la salud, la educacin y la riqueza. El valor va de 0 a 1 siendo cuan ms alto mejor.
  • gdp_for_year ($): (Gross domestic product) (PIB), expresa el valor monetario de la produccin de bienes y servicios de un pas
  • gdp_per_capita ($): relacin entre el GDP y la cantidad de poblacin de un pas
  • generation: Nombre de la generacin a la que pertenece un grupo de poblacin

Si hay algo que destaca en el dataset es que esta columna HDI, a pesar de su importancia, presenta muchos valores NA y esto se convierte en un problema para el anlisis que queremos realizar. Si observamos el dataset tambin vemos que los datos se presentan cada 5 aos, y es en esos huecos en los que aparece NA. Como tcnica de tratamiento de los datos hemos optado para esos huecos rellenarlos con los valores medios a partir de los datos existentes anterior y posterior para cada uno de los pases. Aunque existen en R diferentes libreras para la realizacin de este tipo de acciones, hemos optado por un desarrollo ad-hoc que se ajuste perfectamente a nuestras necesidades. La funcin es la siguiente:

completeHDI_For_YearDatoPosteriorMedia <- function(df){
        j <- 0;
        country <- "";
        vectorNA = c();
        lastValue <- NA
            
        for(i in 1:nrow(df)) {
            
            if(i> 1 && (df[i-1,]$country !=df[i,]$country)) {
                j <- 0;
                vectorNA = c();
                lastValue <- NA
            }

            
            if(is.na(df[i,]$HDI_for_year)) {
                vectorNA <- c(vectorNA,i);
                j <- j+1;
            } else {

                if(is.na(lastValue)) {
                    lastValue <-df[i,]$HDI_for_year
                }
                
                if(length(vectorNA)>0) {
                    for(value in vectorNA) {
                        hdi <- df[i,]$HDI_for_year
                        if(!is.na(lastValue)) {
                            hdi = (lastValue + df[i,]$HDI_for_year)/2
                            
                        }

                        df[value,]$HDI_for_year <- hdi;
                        vectorNA = c();
                        
                        j<-0;
                    }
                    lastValue <- df[i,]$HDI_for_year
                }
            }
        }
        return (df);
        
    }
